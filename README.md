
ioVox

Basic voxel mesh generator in C#.

Class that manages Voxel info.  Data is stored in "compressed" strips.

Constants / Helpers / Debug


ioVox / ioVoxChunk.cs
Manages Chunks.


ioVox / ioVoxel.cs
Voxel management.  Including loading custom type data from XML file.
